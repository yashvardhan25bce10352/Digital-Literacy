## Phishing Attack Case Study

Phishing is a cybercrime where attackers trick users into giving personal information like passwords or bank details.

In a typical case, a student receives an email saying "Your bank account will be blocked". The email contains a fake link. When the user clicks it, it opens a website similar to the bank. The user enters details and attacker gets access.

Students and normal users are main targets because they are less aware.

The consequences include money loss and data theft.
